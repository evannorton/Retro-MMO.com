FAQ

  • When was the game first released?

    August 14, 2020.

  • How long will you continue to release updates for the game?

    Forever.

  • Will savefiles or leaderboards ever be wiped?

    No.

  • Is the game truly free?

    All game content is and always will be free. There will never be anything sold for real world currency which provides any sort of advantage in the game. You can read about the benefits that our optional monthly subscription provides in-game.

  • How can I support the game?

    Being an active member of the community is the biggest help you can give! Financially speaking, purchasing an in-game subscription is the most beneficial way to contribute. You can also make a one-time donation via PayPal or donate monthly via Patreon.

  • How can I contribute to making the game?

    RetroMMO is not currently hiring for any full-time or part-time roles. Anyone can contribute graphics or audio by making a submission in our Discord server. If we are interested in using a submitted asset, we will never use it without reaching out to the author and agreeing on compensation ahead of time. Please consult our style guide before submitting assets.

  • What technologies were used to build the game?

    The key pieces of the tech stack are HTML5, Typescript, PixiJS, NodeJS, Express, SocketIO, MySQL, Sequelize, Heroku, VS Code, Tiled.

  • Is the game open source?

    The server is not, but the client is on GitHub.
